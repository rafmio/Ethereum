AES
Advanced Encryption Standard - симметричный алгоритм блочного шифрования. Размер блока
128 бит, ключ - 128/192/256 бит. Принят в качестве стандарта правительством США.

Block -      последовательность бит, из которых состоит input, output, State, Round Key.
      	     Также под block можно подразумевать последовательность байтов

Cipher Key - секретный криптограф. ключ, кот.используется Key Expansion процедурой,
	           чтобы произвести набор ключей для раундов (Round Keys)
	           Может быть представлен как прамоугольный массив байтов, имеющий 4 строки и Nk колонок

Ciphertext - выходные данные алгоритма шифрования

Key Expansion - процедура герерации Round Keys из Cipher Key

Round Key - Round Keys получаются из Cipher Key использованием процедуры Key Expansion

State - промежуточный результат шифрования, который может быть представлен как
        прямоугольный массив байтов, имеющий 4 строки и Nb колонок


AES - стандарт, основанный на алгоритме Rijndael.
Для AES длина input (блока входных данных) и State (состояния) постоянна и равна 128 бит,
а длина ключа составляет 128, 192 bkb 256 бит.

Вместо длинных рядов бит или байтов, как в большинстве алгоритмов, в AES элементы
организованы в виде матрицы 4 * 4 по 128 бит, т.е. есть сообщение размером
128 бит или 16 байтов в виде матрицы 4 * 4
